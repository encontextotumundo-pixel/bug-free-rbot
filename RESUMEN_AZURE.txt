│  PASO 1: Actualizar .env                                                    │
│  ────────────────────                                                       │
│  Cambiar:  PORT=3000    →    PORT=8080                                     │
│  Cambiar:  NODE_ENV=development  →  NODE_ENV=production                    │
│                                                                             │
│  Archivo: backend/.env                                                      │
│                                                                             │
│                                                                             │
│  PASO 2: Crear App Service en Azure                                        │
│  ──────────────────────────────────                                        │
│  1. Ve a portal.azure.com                                                  │
│  2. Busca "App Service"                                                    │
│  3. Crea uno nuevo                                                         │
│  4. Selecciona:                                                            │
│     - Nombre: mi-sms-xyz                                                   │
│     - Runtime: Node 18 LTS                                                 │
│     - Región: Elige la más cercana                                         │
│     - Plan: B1 (mínimo, $15/mes)                                           │
│                                                                             │
│                                                                             │
│  PASO 3: Desplegar tu código                                               │
│  ──────────────────────────────                                            │
│  Opción A: ZIP Upload (más simple)                                         │
│  ─────────────────────────────────                                         │
│  1. Comprime todo el proyecto                                              │
│  2. En Azure → App Service → Centro de implementación                       │
│  3. Sube el ZIP                                                            │
│  4. Espera a que se extraiga y instale                                     │
│                                                                             │
│  Opción B: Git Push (más profesional)                                      │
│  ──────────────────────────────────────                                    │
│  1. git init                                                               │
│  2. git add .                                                              │
│  3. git commit -m "SMS"                                                    │
│  4. git remote add azure [URL-de-azure]                                    │
│  5. git push azure master                                                  │
│                                                                             │
│                                                                             │
│  PASO 4: Configurar variables en Azure                                     │
│  ───────────────────────────────────────                                   │
│  En Azure Portal → App Service → Configuración                             │
│  Agrega:                                                                    │
│  ├─ PORT = 8080                                                            │
│  ├─ NODE_ENV = production                                                  │
│  └─ RATE_LIMIT_MAX_REQUESTS = 10                                           │
│                                                                             │
│                                                                             │
│  PASO 5: Prueba                                                            │
│  ─────────────                                                             │
│  Abre en navegador:                                                        │
│  https://mi-sms-xyz.azurewebsites.net/health                               │
│                                                                             │
│  Deberías ver:                                                             │
│  {"status":"ok","timestamp":"..."}                                         │

┌─ RESUMEN DE COMANDOS ───────────────────────────────────────────────────────┐
│                                                                             │
│  1️⃣  Cambiar .env                                                          │
│      backend/.env: PORT=8080, NODE_ENV=production                          │
│                                                                             │
│  2️⃣  Comprimir proyecto                                                    │
│      Windows: Compress-Archive -Path * -DestinationPath app.zip            │
│      Mac/Linux: zip -r app.zip . -x "node_modules/*"                       │
│                                                                             │
│  3️⃣  Crear App Service (CLI)                                              │
│      az group create --name sms --location eastus                          │
│      az appservice plan create --name sms-plan --resource-group sms --sku B1
│      az webapp create --name mi-sms-xyz --resource-group sms \             │
│        --plan sms-plan --runtime "NODE|18-lts"                             │
│                                                                             │
│  4️⃣  Desplegar                                                             │
│      az webapp deployment source config-zip --name mi-sms-xyz \            │
│        --resource-group sms --src app.zip                                  │
│                                                                             │
│  5️⃣  Prueba                                                                │
│      curl https://mi-sms-xyz.azurewebsites.net/health                      │
